# 项目说明

## 仓库定位

本仓库是 NBlog 博客系统的 fork 学习仓库，用于参考博客系统的前后端分离实现、后台管理、接口设计和部署方式。

## 技术栈

- 后端：Java、Spring Boot、Maven、MySQL 等
- 前端：Vue / 前端工程化相关技术
- 项目形态：博客前台 + 管理后台 + 后端 API

## 内容结构

- `blog-api/`：后端 API 服务。
- `blog-cms/`：博客管理端。
- `blog-view/`：博客展示前台。
- `deploy.sh`：部署脚本。
- `README.md`：原项目说明文档。

## 学习重点

- 博客系统模块拆分
- API 服务设计
- 前台展示与后台管理分离
- Vue 项目组织方式
- 博客发布、分类、标签、评论等业务流程
- 部署脚本和线上运行方式

## 运行说明

运行前需要准备 JDK、Maven、Node.js、MySQL 等环境。后端需要配置数据库连接，前端需要进入对应目录安装依赖并启动或构建。具体命令以原项目 README 为准。
